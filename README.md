# OLDMYN — Under Construction Experience

Hyper‑cinematic, fully animated under‑construction landing page for the premium streetwear brand **OLDMYN**.

## Run locally

This is a static site. You can open `index.html` directly in a browser, or serve the folder with any static server, for example:

```bash
cd "/Users/tanisbedia/Coming Soon"
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.

## Tech & motion

- **Stack**: Vanilla HTML, CSS, JavaScript
- **Animation**: CSS keyframes + [GSAP 3](https://greensock.com/gsap/) via CDN
- **Features**:
  - Layered parallax construction site with cranes, JCB, workers, drones and particles
  - Cinematic hero with welding, dust, skyline and suspended OLDMYN logo
  - Glowing countdown timer styled like industrial LEDs
  - Scroll‑triggered cards for micro‑moments
  - Industrial caution‑tape strip and premium, streetwear‑inspired UI
